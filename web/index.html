    Browser Not Supported

    The firmware installer requires the Web Serial API, available in Chrome or Edge on macOS, Windows, Linux, or ChromeOS.

    Safari, Firefox, and all mobile browsers do not currently support Web Serial. You can still use the Library Workspace below to convert books.
