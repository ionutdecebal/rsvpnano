# Web Flasher Artifacts

This directory contains the browser-flasher image plus the app-only OTA binary:

- `rsvp-nano.bin` / `rsvp-nano-ota.bin` for the default Waveshare 3.49B rev1 profile.
- `rsvp-nano-rev2.bin` / `rsvp-nano-rev2-ota.bin` for the alternate rev2 profile with GPIO42 backlight control.

For local exports, it is generated by:

```sh
python3 tools/export_web_firmware.py
```

The `.bin` files without a revision suffix are the default release artifacts. The rev2 artifacts are
for boards where the default firmware boots but brightness/backlight control does not respond.

The generated `.bin` files are intentionally ignored by Git. In GitHub Pages deployments, the
site downloads these files from the latest published GitHub Release so the hosted web flasher only
serves official release builds.
